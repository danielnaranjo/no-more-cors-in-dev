
# NO MORE CORS IN DEV

## How to use

Download or [clone the repo](https://github.com/danielnaranjo/no-more-cors-in-dev):

```bash
curl http://fuckcors.now.sh/endpoint={URL}
```

Install it and run:

```bash
npm install
npm run start
```

Deploy it to the cloud with [now](https://zeit.co/now) ([download](https://zeit.co/download))

```bash
now
```
